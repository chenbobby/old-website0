# Data Storage Hierarchy

| Storage Type    | Speed   | Capacity            |
|-----------------|---------|---------------------|
| CPU Register    | 1 ms    | 8 bytes             |
| CPU L1-L3 Cache | 10 ms   | 64 bytes            |
| Main Memory     | 100 ms  | 1,000,000,000 bytes |
| Disk            | 1000 ms | 100 TB              |
